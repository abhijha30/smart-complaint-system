# Smart Complaint Management System

## 📌 Overview
A web-based complaint management system where users can submit complaints and track their status. Admins can manage and resolve complaints.

---

## 🚀 Features

- User Registration & Login
- Admin Login
- Submit Complaint (Water, Electricity, Cleaning, etc.)
- Track Complaint Status
- Admin Dashboard
- Filter Complaints (Date & Type)
- Download Excel Report
- Email Notification with Tracking Link

---

## 🛠 Tech Stack

- Frontend: HTML, CSS, JavaScript
- Backend: Python (Serverless)
- Database: Supabase (PostgreSQL)
- Deployment: Vercel
